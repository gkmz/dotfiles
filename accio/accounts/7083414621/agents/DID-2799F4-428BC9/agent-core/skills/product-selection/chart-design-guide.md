# Chart Design Guide（选品趋势/对比图）

选品环节中若需做趋势、占比、多维度对比等可视化，可参考本指南。阅读后应在后续步骤中编写并执行代码生成图表。

## 工具选择

**原则**：优先用 Seaborn，需要时用 Matplotlib 补充。

| 图表类型 | 推荐方法 | 说明 |
|----------|----------|------|
| 趋势线 | `sns.lineplot()` | 时间序列、搜索/销量趋势 |
| 柱状图 | `sns.barplot()` | 品类对比、增长率 |
| 饼图/环形图 | `plt.pie()` | 占比、市场结构 |
| 多维度对比 | `plt.polar()` | 多属性雷达图 |
| 分布/相关 | `sns.scatterplot()` / `sns.histplot()` | 价格分布、相关性 |

## 推荐用法（Seaborn v0.12+）

使用 `palette` 时需指定 `hue`，避免弃用警告：

```python
# ✅ 正确
sns.barplot(data=df, x='value', y='category', hue='category', palette='viridis', legend=False)
```

## 图表类型与数据用途

| 数据用途 | 推荐图表 |
|----------|----------|
| 时间序列（趋势） | 折线图 + 面积填充 |
| 占比/结构 | 饼图或环形图 |
| 品类对比 | 横向柱状图 |
| 多维度（3+ 指标） | 雷达图（polar） |
| 相关性 | 散点图 |
| 构成随时间变化 | 堆叠面积图 |

## 简洁与合并

- **合并**：相关曲线放在同一张图便于对比，不要每个品类单独一图。
- **避免 trivial**：数据点很少或对比简单时，用表格或文字代替图表。
- **标题**：标题体现结论（如「XX 品类 Q4 增速 +150%」），而非仅写「增长图」。

## 依赖

- `matplotlib`、`seaborn`（若运行环境支持）
